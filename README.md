# simpleblog
纯前端的博客系统,不需要后台服务支持

# 特点
```
基于hexo
基于Node.js
markdown写作,自动解析
一键化部署到服务器
```
# 架构
```
 
simpleblog
│  ├─scaffolds -- 配置文件
│  ├─source    -- 博客写作
│  │  ├─_posts/* -- 样例
│  ├─themes -- 主题模板
```
